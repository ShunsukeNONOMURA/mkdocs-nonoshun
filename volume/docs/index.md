# はじめに
このサイトは主に管理人が勉強したことについてアウトプットしていく場である。  
2023年現在、これまでのアウトプットの整理も兼ねているので、書きかけのページが存在する。  
時間を作って少しずつでも整理を進めていく。  

## 基本事項
- 予告なしに本サイトは停止される場合がある
- 普遍的知見についての記述がメインである
- 積極的な情報発信を行うことを目的としない
- 特筆するトピックについては別リポジトリに分けることにした。リンク参照。
